Download


Para fazer o download do cbuild você pode escolher uma das seguintes opções:

Windows x64

Arquivo ZIP: cbuild-1.0-windows-x64.zip

Linux x64

Arquivo DEB: cbuild-1.0-linux-x64.deb

Arquivo TAR.GZ: cbuild-1.0-linux-x64.tar.gz

O código fonte

Github: https://github.com/italolab/cbuild

Arquivo ZIP: cbuild-1.0.zip

Arquivo TAR.GZ: cbuild-1.0.tar.gz

Como instalar?

Instalando no windows

Para instalar no windows, você pode extrair a versão zipada: cbuild-x.x-windows-x64.zip (onde x.x é a versão) e fazer a variável de ambiente PATH apontar, também, para o diretório extraído (ou onde o arquivo executável "cbuild.exe" estiver.)

Caso queira utilizar a biblioteca cbtest, basta copiar as pastas "lib" e "include" para raiz de onde estiver instalado o mingw ou semelhante.

Instalando no linux

Se estiver no Ubuntu, ou outra distro derivada do Debian, pode fazer o download do pacote .deb: cbuild-x.x-linux-x64.deb (Onde x.x é a versão) e executar o seguinte comando:

sudo dpkg -i cbuild-x.x-linux-x64.deb

Se preferir a versão .tar.gz, basta baixar o arquivo: cbuild-x.x-linux-x64.tar.gz, descompactar, e copiar o executável "cbuild" para pasta /usr/bin ou criar um link simbólico nessa pasta para o executável "cbuild".

Caso queira utilizar a biblioteca cbtest, você pode copiar o arquivo "libcbtest.a" para pasta /usr/lib/x86_64-linux-gnu (isso caso esteja num derivado do debian), e copiar o conteúdo da pasta "include" para pasta /usr/include.

Você pode optar por não instalar o cbtest e incluí-lo diretamente no seu projeto. Veja como fazer essa configuração em: testes com cbtest.

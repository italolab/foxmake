Download


Para fazer o download do foxmake você pode escolher uma das seguintes opções:

Windows x64

Arquivo ZIP: foxmake-1.2-windows-x64.zip

Linux x64

Arquivo DEB: foxmake-1.2-linux-x64.deb

Arquivo TAR.GZ: foxmake-1.2-linux-x64.tar.gz

O código fonte

Repositório : https://github.com/italolab/foxmake

Arquivo ZIP: foxmake-v1.2.zip

Arquivo TAR.GZ: foxmake-v1.2.tar.gz

Notas da versão 1.2

Esta versão traz melhorias sobre a compilação concorrente dos arquivos de código fonte. Na versão 1.0, eram alocadas tantas threads quanto fossem os arquivos de código fonte. Nesta versão, são alocadas threads conforme o número de núcleos do processador, melhorando o desempenho da compilação e, também, a exibição na saída padrão.

Como instalar?

Instalando no windows

Para instalar no windows, você pode extrair a versão zipada: foxmake-x.x-windows-x64.zip (onde x.x é a versão) e fazer a variável de ambiente PATH apontar, também, para o diretório extraído (ou onde o arquivo executável "foxmake.exe" estiver.)

Caso queira utilizar a biblioteca xutest, basta copiar as pastas "lib" e "include" para raiz de onde estiver instalado o mingw ou semelhante.

Instalando no linux

Se estiver no Ubuntu, ou outra distro derivada do Debian, pode fazer o download do pacote .deb: foxmake-x.x-linux-x64.deb (Onde x.x é a versão) e executar o seguinte comando:

sudo dpkg -i foxmake-x.x-linux-x64.deb

Se preferir a versão .tar.gz, basta baixar o arquivo: foxmake-x.x-linux-x64.tar.gz, descompactar, e copiar o executável "foxmake" para pasta /usr/bin ou criar um link simbólico nessa pasta para o executável "foxmake".

Caso queira utilizar a biblioteca xutest, você pode copiar o arquivo "libxutest.a" para pasta /usr/lib/x86_64-linux-gnu (isso caso esteja num derivado do debian), e copiar o conteúdo da pasta "include" para pasta /usr/include.

Você pode optar por não instalar o xutest e incluí-lo diretamente no seu projeto. Veja como fazer essa configuração em: testes com xutest.
